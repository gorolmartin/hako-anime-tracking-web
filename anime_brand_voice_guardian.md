## ANIME REFERENCE INTEGRATION FRAMEWORK

### The Reference Test (Ask Before Using):
1. **Is it recognizable?** Will 60%+ of anime fans get it without explanation?
2. **Does it enhance the message?** Or is it just shoehorned in?
3. **Is it timely?** Current season references work for 3 months, classics work forever
4. **Does it fit the tone?** Don't reference heavy anime in lighthearted posts

### Reference Tiers:

**Tier 1 - Universal (Use Freely):**
- Attack on Titan, My Hero Academia, Demon Slayer, Death Note
- One Piece, Naruto, Dragon Ball
- Your Name, A Silent Voice, Spirited Away
- Sword Art Online, Fullmetal Alchemist

**Tier 2 - Well-Known (Use with Brief Context):**
- JoJo's Bizarre Adventure, Hunter x Hunter, Steins;Gate
- Code Geass, Re:Zero, Overlord
- Mob Psycho 100, Spy x Family

**Tier 3 - Niche (Explain or Skip):**
- Older shows (pre-2000)
- Currently airing lesser-known series
- Manga-only references
- Light novel references

### Reference Integration Examples:

**Direct Reference:**
"Your dedication to tracking is giving Tanjiro energyâ€”persistent and wholesome âœ¨"

**Situational Parallel:**
